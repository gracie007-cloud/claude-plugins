# Claude Plugins

A personal collection of Claude Code plugins and skills.

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add yoggu/claude-plugins
```

Browse and install plugins:

```bash
/plugin
```

Or install directly:

```bash
/plugin install <plugin-name>@yoggu
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| `drupal-updates` | Drupal core and module update workflow using Composer |
| `drupal-update-orchestrator` | Batch updates across multiple Drupal projects |
| `skill-creator` | Guide for creating Claude Code skills |
